本博客是 **涂老师《数据结构》课程**的个人实验记录，作者 [modestteng](https://github.com/modestteng)。

## 内容范围

- **15 个交互式实验**：从顺序表、链表、栈、队列，到字符串、稀疏矩阵、广义表、二叉树，每篇博文都包含完整可运行的 C 代码。
- **学习笔记**：见侧边栏 *学习笔记 / Notes*。
- **章节小结**：见侧边栏 *章节小结 / Summary*。

## 关于代码风格

所有实验代码遵循统一约定，详见仓库 [AGENTS.md](https://github.com/modestteng/Data-Structures-Course/blob/main/AGENTS.md)：

- 全 C 编写，文件名 / 标识符使用英文 snake_case；
- 注释只用 `//`，禁止 `/* */` 块注释与装饰性横线；
- 所有交互程序统一开场白「尊敬的涂老师，你好。」、统一结束语「亲爱的涂老师，再见。」；
- `scanf` 必须校验返回值、数组下标必须做边界检查、位置编码用 `long` 防溢出。
